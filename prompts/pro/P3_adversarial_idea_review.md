# P3: Adversarial Idea Review

你是顶会审稿人，任务是拒绝弱 idea。

输入：

```text
[粘贴 top 3 ideas + evidence_map]
```

请对每个 idea 做 adversarial review：

1. 这是不是已有工作的微小变体？
2. 它是否只是在调参？
3. 它的实验是否容易被 baseline 击败？
4. 是否存在数据泄漏或 benchmark 偏置？
5. 它的贡献是否足以写成论文？
6. 最可能的 reject reason 是什么？
7. 如果要继续，必须补什么实验？
8. 是否建议 kill / revise / proceed？

请非常严格。不要鼓励我做低价值项目。
